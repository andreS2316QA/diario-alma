
📖 Diario Alma

Diario Alma es una aplicación web diseñada para el bienestar personal y la reflexión diaria. Combina una interfaz limpia con una lógica de servidor en Python para ofrecer un espacio seguro donde los usuarios pueden gestionar sus pensamientos y estados de ánimo.
✨ Características Principales

    Gestión de Entradas: Permite la creación y visualización de notas personales.

    Identidad Visual: Cuenta con un icono personalizado (imagen.ico) para una experiencia de marca cohesiva.

    Ambiente Sonoro: Integración de recursos de audio (la_reina.mp3) para acompañar la escritura o la meditación.

    Servidor Robusto: Desarrollado sobre un micro-framework que garantiza rapidez y estabilidad.

🛠️ Stack Tecnológico

    Lenguaje: Python 3.13

    Framework Web: Flask (especificado en las dependencias del proyecto).

    Frontend: HTML5 y CSS3 para la estructura y el diseño visual.

    Multimedia: Soporte nativo para audio MP3.
